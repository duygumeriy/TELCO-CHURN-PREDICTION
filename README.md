# TELCO CUSTOMER CHURN PREDICTION

This project focuses on predicting customer churn in a telecommunications company using machine learning techniques.

## Project Goal
The main objective of this project is to identify customers who are likely to leave the company and help businesses take proactive actions to reduce churn.

## Dataset
- Telco Customer Churn Dataset
- Includes customer demographics
- Service subscriptions
- Contract information
- Monthly charges
- Total charges
- Churn status

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Handling Class Imbalance
5. Model Training
6. Model Evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score

## Key Insights
- Customers with month-to-month contracts are more likely to churn.
- Higher monthly charges increase churn probability.
- Long-term contract customers are less likely to leave.
